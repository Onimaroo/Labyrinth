# Labyrinth
Un générateur de labyrinthe fait en C. Un mini-projet scolaire fait avec un binôme.

## Comment l'éxécuter?
Il faut simplement lancer le fichier `Labyrinth.c` sur le terminal après s'être placé dans la racine du projet.
